== Description ==

This is plugin for creating a widget for recent post. In it user can select number of posts to be displayed in front end.It shows post's featured image along with post title.

This Widget is created for Get Recent post.

== Features ==

1. Show Recent post.
2. Show Date with post.
3. Show Author with post.
4. Show Category with post.
5. Show post featured image.


== Installation ==
1. Unzip and upload the `Recent-post-widget` directory to the plugin directory ('/wp-content/plugins/')
2. Activate the plugin through the 'Plugins' menu in WordPress
